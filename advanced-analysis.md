# Advanced Analysis

WaveBPS can do advanced analysis. [When a data base is enabled](wavebps-basic-databases.md) you can do statistics on the entire capture according to node (Figure 1). WaveBPS will calculate the min, max, mean for every automatic measurement organized by node. An example node analysis performed on the "dwcan diff traffic" example waveform is [included here](https://791501338-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FZRYtbPKu55JGqk9BcS9C%2Fuploads%2FZOytAfy2jdN6uLB5RAHt%2FNodeAnalysis.png?alt=media\&token=5c61e86d-313a-43fc-8c52-1961ba5899e0) . This type of analysis quickly helps you find nodes that are acting badly.
